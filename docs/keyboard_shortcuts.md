# Keyboard Shortcuts

These shortcuts help you work faster.

## General

| Shortcut | Action |
| --- | --- |
| F1 | Open User Guide / Manual |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+Shift+Z | Redo |

On macOS, standard system shortcuts may use Cmd instead of Ctrl.

## Workspace Layout

| Shortcut | Action |
| --- | --- |
| Ctrl+1 | Toggle Left Panel |
| Ctrl+2 | Toggle Right Panel |
| Ctrl+` | Toggle Focus Preview |
| Ctrl+Alt+0 | Reset Layout |

## Subtitle Editing

These shortcuts work when the Subtitle Editor is focused.

| Shortcut | Action |
| --- | --- |
| Alt+Left | Move selected cue 0.1 seconds earlier |
| Alt+Right | Move selected cue 0.1 seconds later |
| Alt+Shift+Left | Move selected cue start 0.1 seconds earlier |
| Alt+Shift+Right | Move selected cue end 0.1 seconds later |
| Ctrl+/ | Split selected cue at the current preview time |
| Ctrl+M | Merge selected cue with next cue |
| Ctrl+Shift+M | Merge selected cue with previous cue |

## Preview Zoom

Use the Preview zoom dropdown to choose:

- Fit
- 10%
- 20%
- 25%
- 50%
- 75%
- 100%
- 125%
- 150%
- 200%

If Ctrl + mouse wheel is enabled in your preview area, it follows these same zoom steps.
